# Minecraft Mod Manager

A tool to download mods from curseforge by terminal.

## Usage

```shell
> mcmm -h
NAME:
   Minecraft Mod Manager - MCMM for windows/amd64

USAGE:
   mcmm [global options] command [command options] [arguments...]

VERSION:
   0.0.1

AUTHOR:
   SteveXMH

COMMANDS:
   search, s    search for a mod id by name
   download, d  Download a mod with dependcises from curseforge
   info         Get mod info and version compatibility list.
   help, h      Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```
